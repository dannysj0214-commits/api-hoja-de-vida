# API de Experiencias Profesionales

API RESTful para gestionar experiencias laborales en una hoja de vida.

## Tecnologías utilizadas
- Node.js
- Express.js
- MongoDB (Mongoose)
- Postman

## Endpoints de la API

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/api/experiencias` | Obtener todas las experiencias |
| GET | `/api/experiencias/:id` | Obtener una experiencia |
| POST | `/api/experiencias` | Crear nueva experiencia |
| PUT | `/api/experiencias/:id` | Actualizar experiencia |
| DELETE | `/api/experiencias/:id` | Eliminar experiencia |

## Cómo ejecutar el proyecto

```bash
npm install
npm run dev


 el servidor correra en : http://localhost:3001
